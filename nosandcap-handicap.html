How It Works

1

Score Differential

Computed per round using gross score, course rating, and slope.

Diff = (Score − Rating) × (113 ÷ Slope)
2

Rolling 4-round window

Each group tracks the most recent 4 rounds per player. A 5th round pushes out the oldest. Missed weeks are simply not entered.

3

Handicap Index

Index = (Avg of all diffs) × 0.96 [truncated]
4

Course Handicap

Course Hdcp = round( Index × (Slope÷113) + (Rating−Par) )

Import GHIN indexes to start the season

One-time season setup. Enter each player's current GHIN Handicap Index below. The app will back-calculate a synthetic differential and store it as their Round 1 starting point. After that, the rolling 4-round system takes over automatically.

Only enter indexes for players who don't already have rounds recorded — existing rounds won't be touched.
